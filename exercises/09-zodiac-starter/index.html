Problem 1

Remember the Birthday

Enter a birthday, find the zodiac sign, and save the birthday to localStorage so it pre-fills the next time the page loads.

✅ Loaded your saved birthday!
💡 Hint #1 — Selecting elements

Select the form, input, output, and badge at the top of your script:

const rememberForm = document.querySelector('#remember-form');

const birthday2Input = document.querySelector('#birthday2');

const rememberOutput = document.querySelector('#remember-output');

const storageBadge = document.querySelector('#storage-badge');

💡 Hint #2 — Parsing the date & getting the sign

The input's value is a string like "1995-07-14". Convert it and pull out month + day:

const date = new Date(dateValue);

const month = date.getMonth() + 1; // getMonth() starts at 0!

const day = date.getDate();

Then call the helper: const sign = getZodiacSign(month, day);

💡 Hint #3 — Saving to localStorage on submit

Inside your submit handler, after you get the date value, save it:

localStorage.setItem('savedBirthday', dateValue);

Use the raw string value from the input — not a Date object.

💡 Hint #4 — Loading on page open

Outside (above) your event listener, check if a value was saved:

const saved = localStorage.getItem('savedBirthday');

if (saved) {

  birthday2Input.value = saved;

  storageBadge.style.display = 'block';

}

To test: submit a birthday, then refresh — the input should pre-fill and the green badge should appear.

Problem 2

Delayed Alert with setTimeout

Enter a birthday and hit the button. Use setTimeout to pop up a browser alert() showing the zodiac sign after 5 seconds.

💡 Hint #1 — Get the sign first

Parse the date value and call the helper to get the sign — same as Problem 3:

const date = new Date(birthday3Input.value);

const month = date.getMonth() + 1;

const day = date.getDate();

const sign = getZodiacSign(month, day);

💡 Hint #2 — Using setTimeout

setTimeout runs a function after a delay (milliseconds). Put your alert() inside it:

setTimeout(() => {

  alert(`Your sign is ${sign}!`);

}, 5000); // 5000ms = 5 seconds

💡 Hint #3 — Cancelling with clearTimeout

If the user submits again before the 5 seconds are up, cancel the old timer so two alerts don't fire:

let signTimer; // declared outside the event listener

clearTimeout(signTimer); // cancel any running timer

signTimer = setTimeout(() => { alert(...); }, 5000);

Problem 3

Find Your Zodiac Sign

Enter a birthday and click the button to see the zodiac sign. The formula is already written for you — just hook it up!

💡 Hint #1 — Selecting elements

Select the form, the date input, and the output div at the top of your script:

const zodiacForm = document.querySelector('#zodiac-form');

const birthdayInput = document.querySelector('#birthday');

const zodiacOutput = document.querySelector('#zodiac-output');

💡 Hint #2 — Listening for submit

Attach a submit listener to the form. The very first line inside must prevent the default page reload:

zodiacForm.addEventListener('submit', (event) => {

  event.preventDefault();

});

💡 Hint #3 — Parsing the date

The input's value is a string like "1995-07-14". Convert it and pull out month + day:

const date = new Date(birthdayInput.value);

const month = date.getMonth() + 1; // getMonth() starts at 0!

const day = date.getDate();

💡 Hint #4 — Calling the formula & showing the result

The getZodiacSign function is already at the bottom of app.js. Pass it month and day:

const sign = getZodiacSign(month, day);

Then display it:

zodiacOutput.textContent = `Your sign is ${sign}!`;
